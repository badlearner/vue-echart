# vue-echart
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```
# mock静态数据服务开启 执行
npm run mock

# mock静态数据服务开启和dev 同时执行
npm run mockdev
